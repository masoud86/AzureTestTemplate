Overview

This REST API allows developers to interact with the MyApp platform. You can manage users, view statistics, and perform CRUD operations on your data resources.

Base URL: 
https://api.myappdemo.com/v1

Authentication

All endpoints require a valid Bearer Token in the request header: 

Authorization: Bearer <your_token_here>

Endpoints

GET /users List all users

Retrieves all users from the system.

curl -X GET https://api.myappdemo.com/v1/users \
  -H "Authorization: Bearer <your_token>"

Response:

[
  {
    "id": 1,
    "name": "John Doe",
    "email": "john@example.com"
  }
]
POST /users Create new user

Creates a new user with the provided data.

curl -X POST https://api.myappdemo.com/v1/users \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer <your_token>" \
  -d '{"name": "Jane Doe", "email": "jane@example.com"}'

Response:

{
  "id": 2,
  "name": "Jane Doe",
  "email": "jane@example.com"
}
